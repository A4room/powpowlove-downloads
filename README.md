# MoaiWannaSlam Downloads

Static Android APK download page for `모아이는내려찍고싶다` (`MoaiWannaSlam`).

The public page shows only `prod` records from `releases.json`. Internal `dev` records live in `dev-releases.json` as an encrypted manifest unlocked by the dev key. This prevents casual public access to dev links, but strong access control still requires hosting dev APKs somewhere private or authenticated.
